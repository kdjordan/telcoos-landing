# telcoos-landing

Marketing landing page for [telcoos.io](https://telcoos.io).

A single self-contained `index.html` — no build step, no framework. Brand fonts
(Unbounded + Inter) load from Google Fonts; everything else is inline. An nginx
`Dockerfile` is included for container-based static hosting.

## Editing

It's one file. Edit `index.html` and redeploy.
